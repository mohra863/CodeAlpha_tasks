# Object Detection and Tracking

## Description
A real-time Object Detection and Tracking application built with Streamlit, YOLOv8, and the SORT tracking algorithm. The application allows users to upload images or videos and detect multiple objects while tracking them across frames.

## Features
- Real-time object detection using YOLOv8.
- Multi-object tracking using the SORT algorithm.
- Upload images and videos.
- Interactive web interface with Streamlit.
- Displays bounding boxes and object IDs.

## Technologies Used
- Python
- Streamlit
- OpenCV
- YOLOv8 (Ultralytics)
- NumPy
- SciPy
- FilterPy
- SORT Tracking

## Installation

```bash
pip install -r requirements.txt
```

## Run the Application

```bash
streamlit run app.py
```
