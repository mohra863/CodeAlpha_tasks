# CodeAlpha Tasks

This repository contains the projects I completed during the CodeAlpha AI Internship.

## Projects

### 1. Language Translation Tool
- Translates text between multiple languages.
- Built with Streamlit and Deep Translator.
- Supports Text-to-Speech using gTTS.

### 2. Object Detection and Tracking
- Real-time object detection using YOLOv8.
- Multi-object tracking using the SORT algorithm.
- Built with Streamlit, OpenCV, and Ultralytics.

### 3. AI & Machine Learning FAQ Chatbot
- FAQ chatbot based on Sentence Transformers.
- Answers AI & Machine Learning questions using semantic similarity.
- Built with Streamlit.

## Technologies Used
- Python
- Streamlit
- OpenCV
- YOLOv8
- Deep Translator
- gTTS
- Sentence Transformers
- NumPy
- SciPy
- FilterPy
