# AI & Machine Learning FAQ Chatbot

## Description
A Streamlit-based FAQ chatbot that answers questions related to Artificial Intelligence and Machine Learning using Sentence Transformers for semantic similarity.

## Features
- Interactive chatbot interface.
- Answers AI & Machine Learning FAQs.
- Semantic search using Sentence Transformers.
- Fast and user-friendly web application.

## Technologies Used
- Python
- Streamlit
- Sentence Transformers
- PyTorch

## Installation

```bash
pip install -r requirements.txt
```

## Run the Application

```bash
streamlit run app.py
```
